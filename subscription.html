1. 安装并运行

⚠️
重要说明：改版会自动适配路径，主要解决线路稳定性问题，请放心使用。

1.1 系统要求

✓ 操作系统：macOS 10.15+ / Ubuntu 20.04+ / Debian 10+ / Windows
✓ 硬件：≥ 2 GB RAM
✓ 软件：Node.js 18+、Git 2.23+（可选）、GitHub/GitLab CLI（可选）
✓ 网络：无需翻墙

1.2 安装CC（改编版）

npm install -g https://gaccode.com/claudecode/install --registry=https://registry.npmmirror.com

-g 表示全局安装，可在任意目录调用 claude

如果已经安装过原版的，可执行如下卸载：

npm uninstall -g @anthropic-ai/claude-code
rm -rf ~/.claude*

⚠️
网络问题解决：如果遇到与cc链接的网络问题，请执行 claude --pick-relay 切换线路试下
# 进入项目目录
cd your-project-folder

# 启动 CC
